## 多模板与样式统一

- 增加模板选择器，并保证以下三处使用同一 `style`：

  - 公开链接页面预览

- 为旧样式参数提供别名映射，避免老链接失效（例如 `sidebar/minimal/mono` 等）。

## PDF/公开链接修复

- 修复 PDF 下载失败与公开链接访问异常：
  - PDF 改为后端 Puppeteer 生成真实 `application/pdf` 二进制。
  - 私有下载不再用新窗口打开（避免丢 JWT Header），改为 `fetch` 后下载。
  - 公共/私有 PDF 都支持 `style` 参数，确保“选中样式 = 导出样式”。
- 增强 PDF 失败时的错误返回与日志记录（便于从 `apps/api/logs/*` 定位）。

## SQLite 只读写入问题（公开链接重生成）

- 处理 `attempt to write a readonly database`：
  - 确认 `apps/api/prisma/dev.db` 与目录权限可写。
  - 重启 API 进程，避免旧连接句柄导致只读报错。

- 明确区分「详情页」与「搜索/导航页」，避免把“加载中/安全校验/导航”文本传给 AI。
