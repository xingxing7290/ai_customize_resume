## PDF 中文方框修复

- 定位原因：服务器仅有 DejaVu/Liberation 等西文字体，缺少 CJK 字体，Chromium 无法渲染中文字形导致方框。
- 服务器安装中文字体并刷新缓存：
  - `fonts-noto-cjk`
  - `fonts-noto-cjk-extra`
  - `fc-cache -fv`
- PDF CSS 字体栈调整为优先使用 `Noto Sans CJK SC`，确保 Puppeteer 生成 PDF 时正确匹配中文字体。
- 验证：生成的 PDF 中已能看到嵌入的 `NotoSansCJKsc-Regular/Bold` 字体信息，中文应不再是方框。
