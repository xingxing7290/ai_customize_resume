# Issues

## Issue 001: JD 链接抓取稳定性

### 问题
招聘网站可能有反爬机制，链接抓取可能失败或不稳定。

### 影响
- 用户输入链接后无法解析
- 用户体验差

### 当前状态
open

### 建议处理方案
1. 保底支持纯文本 JD 输入
2. 抓取失败时提示用户粘贴文本
3. 后续可考虑集成第三方抓取服务

## Issue 002: AI 响应时间

### 问题
AI API 响应可能较慢，影响用户体验。

### 影响
- 简历生成等待时间长
- 用户可能放弃操作

### 当前状态
open

### 建议处理方案
1. 设置合理超时（30s）
2. 异步处理，显示进度
3. 提供重试机制
4. 缓存常见岗位解析结果

## Issue 003: AI 输出格式不稳定

### 问题
大模型输出 JSON 格式可能不稳定，偶尔不符合 Schema。

### 影响
- 解析失败
- 需要重试

### 当前状态
open

### 建议处理方案
1. 使用 Structured Output / JSON Mode
2. 实现 JSON 修复工具
3. 设置重试机制
4. 记录失败日志分析

## Issue 005: 公开链接安全性

### 问题
公开简历链接可能被滥用或泄露。

### 影响
- 简历信息泄露
- 用户隐私风险

### 当前状态
open

### 建议处理方案
1. 使用随机 token，不暴露 ID
2. 提供撤销发布功能
3. 记录访问次数，异常时提醒
4. 后续可设置有效期

## Issue 006: 前后端类型一致性
